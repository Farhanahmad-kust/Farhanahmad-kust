## Featured Projects

### CampaignOS  
**Sales Automation Command Center**

CampaignOS is a multi-agent sales automation system designed to automate the full sales funnel, from lead ingestion to enrichment, outreach, voice calls, and analytics.

Key highlights:

- Built a multi-agent n8n automation ecosystem
- Automated lead ingestion, enrichment, and outreach workflows
- Integrated GPT-4 for personalized email generation
- Integrated Vapi Voice AI for autonomous sales calls
- Synced call transcripts and analytics into a Supabase dashboard
- Designed the system to reduce manual sales operations

---

### Punjab Agri-Sahulat Bot  
**RAG System for Agricultural Support**

Punjab Agri-Sahulat Bot is an AI-powered agricultural assistant designed to provide localized farming guidance to users through a retrieval-augmented generation system.

Key highlights:

- Co-developed a RAG agent using n8n and Pinecone
- Designed support for Urdu-language inputs
- Added voice-to-text support for non-technical users
- Focused on accessibility for rural farming communities
- Built around verified agricultural information retrieval

---

### Legal Contract Analyzer  
**Enterprise Document Intelligence Prototype**

Legal Contract Analyzer is a rapid prototype for analyzing legal and business documents using Azure AI services.

Key highlights:

- Designed an enterprise document analysis workflow
- Used Azure Document Intelligence for document extraction
- Used Azure OpenAI for contract understanding and analysis
- Focused on making complex legal documents easier to review
- Built as a practical AI assistant for document-heavy workflows
